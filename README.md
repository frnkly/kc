# Getting started

Make sure [Yarn](https://yarnpkg.com) is installed on your machine:

```shell
yarn --version
```

Then clone the repository and install the dependencies:

```shell
git clone git@github.com:kwcay/kwcay.co.git
cd kwcay.co
yarn install
```

# Running the app

```shell
# Run the app locally.
yarn start

# Deploy to Github Pages.
yarn deploy
```
